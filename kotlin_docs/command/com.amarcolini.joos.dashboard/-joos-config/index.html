JoosConfig

@Target(allowedTargets = [AnnotationTarget.CLASS, AnnotationTarget.FIELD, AnnotationTarget.PROPERTY])
annotation class JoosConfig(val name: String = "")
Content copied to clipboard

An alternative to Config that automatically handles more complex types like trajectories, poses, and angles, or even custom types using WithConfig. Its functionality is the same such that if a class has this annotation, all of its public, static, non-final fields will be added to FTC Dashboard (unless the class also has the Disabled annotation).
